# KPIゲージ手動検証ガイド

## 検証手順

1. **ブラウザを開く**
   - プレビューサーバー: http://localhost:4173/ValueScope/
   - または開発サーバー: http://localhost:5173/ValueScope/

2. **KPI分析タブをクリック**

3. **各企業のKPIゲージを確認**

### 検証ポイント

#### ✅ 全てのゲージが9時（左）から15時（右）の範囲で表示される
- グレーの背景ゲージが9時から15時まで（左から右へ180度）
- 値を示すカラーバーも同じ範囲内

#### ✅ 全ての値が12時（真上）を超えない
- 最も高い値（中部電力DSCR 15.95倍）でも12時手前（11:59）に収まる
- 他の値も全て12時より左側に表示される

#### ✅ ROEゲージのyellow/greenが9時から始まる
- 以前の問題: yellow/greenが12時から始まっていた
- 修正後: greyと同じように9時から始まり15時で終わる

## 修正内容

### ROE max値: 20% → 32%
- **修正前**: CHUBU 15.26% / max 20% = 76.3% → 137度（1:16）❌ 12時を超える
- **修正後**: CHUBU 15.26% / max 32% = 47.7% → 85.8度（11:51）✅ 12時未満

### 自己資本比率 max値: 80%（維持）
- CHUBU 37.91% / max 80% = 47.4% → 85.3度（11:50）✅ 12時未満

### DSCR max値: 32倍（維持）
- CHUBU 15.95倍 / max 32倍 = 49.8% → 89.7度（11:59）✅ ギリギリ12時未満
